string methods

toUpperCase()
all upcase
toLowerCase()
all lower case
charAt
takes an index number and returns the character found at that position
indexOf()
returns index number of hte first time a character or set of characters is found in a string
lastIndexOf
index number of last time a character or set of characters is found
substring()
returns characters found between 2 index numbers where
  1. first index number includes the character
  2. last index number excludes the character

eg substring(8,14) of "home seet home" will return characters number 8 to 13 ie 'et hom'

split()
spits the string each time the character is found
trim()
removes white spacde from start and end
replace()
like find and replace - syntax replace('me', 'w') - replaces 'me' with 'w'

Syntax uses object dot notation and passes in the string as the object

var saying = 'home sweet home'

eg

saying.length

saying.toUpperCase

saying.substring(8.14)
