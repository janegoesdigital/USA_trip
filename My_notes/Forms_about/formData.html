How do I use data collected from a form?

  • User interface
  • Retrieving and using data

Receiving the data from the forms and using it in javascript is working out what's been selected

So, I need to know what has been selected. It will be selected on click

There are two actions here when using a radio button, There is a click in the radio button, followed by a click on the button to submit the form.

Selecting radio buttons

This will select all the radio buttons with the name=eventType

$( "input[name=eventType]:radio" )

This will get the element

$("input[name=chooseEvent]")

This will get the value selected

$( "input[name=eventType]:checked").val();

In the js, define the returned value as a variable, and then use it in the JS

see jQuery

when the choose event button is clicked, create a variable with the value equal to the radio button with a name of eventType and return the value

Once I know what has been selected, I have different workflows depending on what's been selected

If an eventType with a value of stay has been submitted, I'll show the date picker etc
