How do forms work?

  1. Users fills in a form and presses button to send form to server
  2. Name of each form control is sent to the server along with the value of the control. Forms have key: value pairs known as form controls. There may be a lot of these on one form
  3. Server processes the form using another language and may store the information in a database
  4. Server creates a new page to send back to the browser based on the information received

The form tag

Use the form tag. For every form tag, must have:

  • a method attribute (get or post - usually post if using a DB or have sensitve infor)
  • an action attribute whose value is the URL for the page on the server that will receive the information in the form
