Events

What are events?

Events are the browser saying 'hey - this just happened!'

Events are said to be fired or raised and trigger scripts

  1. user interaction creates an event;
  2. the event triggeres some code;
  3. the code responds to the user somehow

Working with events

  1. html event handlers - not recommended
  2. traditional DOM event handlers
  3. DOM level 2 event listeners - these are good - enable you to trigger multiple functions on one event
  4. jQuery

Event listeners

element.addEventListener('event', functionName [,Boolean]);

eg

function = checkUserName(){};

jQuery events - .on() method

.on() method used to handle all events

    Steps:
  1. use a selector to create a jQuery selector;
  2. use .on() to indicate which event you want to respond to

eg

$('li').on('click', function(){})
