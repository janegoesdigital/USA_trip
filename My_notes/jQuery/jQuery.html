What is jQuery?

Way of manipulating the DOM using CSS-style selectors

Creates an object called jQuery which holds reference to the DOM elements.

Can call the object jQuery() or us the shorthand $().

Has cross-browser suppor and event handler simpler

Once have made a selection from the DOM, can chain methods.

Simpler to loop through elements

jQuery motto - "Write less - do more"

How to include jQuery

Need to include a reference to it in a script tag.

Include the file in your site

Place it just before the closing body tag

CDN

Can use CDN (content delivery network) - series of servers spread out around the world designed to serve static files. Tries to find server near you, and send the file. If include it this way, need to include an OR in case it can't be loaded

Doing something with the selected elements

Getting the content only

.html()

retrieves only the HTML inside the FIRST element selected

.text() method

returns content from every elemenet.

.val() - gets the content from an input or text area

Replacing the content

Can use the .html and .text methods - pass in the as an argument the text you want to add. Will change all of the content of the elements selected

p$('li)').html("update the content")

replaceWith()

replacces every element wiht the new content. Pass in the new content as an argument into the function

remove()

Like it's name said - it removes the selected content.
