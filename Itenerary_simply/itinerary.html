Our American Adventure 2016

About this page and what I've learnt so far

Where is the data coming from?

Currently, I have a very simple js function working. I have stored the information about our trip in an array with objects in that array detailing each leg of the journey. I have then used that array, to display information (in a very unbeautiful format - on the page ). Ultimately, I would like the data to come from our trips collections in the mongoDB

How am I accessing it

So far, I am working with the data about each journey from one place to another. I have found this very challenging! With the initial functions which I wrote to retrieve the individual journeys, I used the ECMA6 forEach loops. This worked well for that particular use case.

The calculating the cost function, I found challenging. I finally solved this by using the following

      a filter function, to select only those things with costs in them
      a map function - to enable me to more readily access the values in the object nested in the array
    ; and
      then the reduce method

How are you getting things on the page?

Currently, I am working with the DOM only to get things on the page.
